வெண்ணிற மணலின் வெப்பந் தணித்து, \
நிலமகளைப் புணரும் பெருங்கடலை, \
மேகக் கூட்டத்துள் தலை நுழைத்து, \
வானத்து இரகசியங்களை ஒட்டுக் கேட்கும் மலையை, \
வேங்கை போலாகிக் கதிரவனும், பதுங்கியே பாயும் காட்டை, \
மலை காடு அலைந்து, \
அனுபவப்பட்டு, \
கடல் வந்து சேரும் நதியை, \
தூரத்து சொந்தங்களின் அணிவகுப்பாய்,\
 இரவு மௌனத்தை கொண்டாட்டமாக்கும் வானத்தை, \
அந்தத்து சாட்ச்சியாய் ஆங்காங்கே குவிந்து, \
அங்கத்தை ஒரு நாள் தழுவத் துடிக்கும் தீயை, \
காலத்தை பகடி செய்து, \
வானமளந்து,\
நிரந்தரத் தவங்கிடக்கும் பெருமரத்தைக் காணுபோதெல்லாம் \
நீ உன்னைக் கண்டிருக்கிறாயா? \
எப்படி இருந்தாய்? \
இருந்தாயா?

— அயலவன்
